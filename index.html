About Me

Born and raised in Kuala Lumpur, Malaysia, I'm currently living in Los Angeles, California while I work on completing my undergraduate studies at CSU-Northridge.

Like many others, my love for video games introduced me to the world of programming. My first experience with code was when I had to mess around with a reverse-engineered Java-based MMORPG in order to spin up a private server. Though I had no idea what I was doing at the time, it was already too late — I got bit by the coding bug and eventually moved on to writing simple CLI-based programs like tic-tac-toe or an arithmetic calculator. Today, I mostly work on web-based projects. Although I'm comfortable with the whole stack, I take particular interest in the back-end and infrastructure side of things.

I believe in using my skillset in programming to give back to society. I've contributed to open-source projects like if-me, a platform for sharing mental health experiences. All my personal projects are also open-source — I've learned a lot through reading others' code, and I hope I can help others do the same by sharing my code. You can check out my portfolio to read about some of the things I've made so far.

Most of my programming experience so far has been designing and developing programs using Ruby, Python and JavaScript. With these languages, I've had experience using front-end libraries such as React and jQuery, backed by server-side frameworks like Ruby on Rails and Flask.
