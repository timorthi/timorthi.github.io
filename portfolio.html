r-raffle

Live demo at r-raffle.co
Technologies used: Python (Flask), JavaScript, HTML/CSS, SQLite, Linux

r-raffle is a simple raffle system for Reddit that aims to help contest organizers with selecting their winner(s).

Given a Reddit submission and a set of user-defined parameters (requirements), the webapp processes the submission and randomly selects a winner/a number of winners based on the aforementioned requirements. Read more...

MonsterInfoBot

Source code available on GitHub.
Technologies used: Python

MonsterInfoBot (retired) was an automated system in Reddit that responds to user calls in a specific subreddit. Given valid user input, the program scrapes an online database for tabular data and displays the data in a Reddit-friendly (read: Markdown) format.
